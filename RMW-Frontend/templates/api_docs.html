API Documentatie

Volledige documentatie van de Gewichtstracker API

Introductie

De Gewichtstracker API biedt toegang tot alle functionaliteiten van de Gewichtstracker applicatie. Met deze API kunt u gewichtsgegevens ophalen en toevoegen, berekeningen uitvoeren en gebruikers beheren.

Basis URL

http://localhost:8000

Authenticatie

Authenticatie verloopt via het /api/login endpoint. Na succesvolle authenticatie ontvangt u een gebruikers-ID dat u kunt gebruiken voor verdere API-verzoeken.

POST /api/login

Request Body:

{
  "username": "test",
  "password": "password"
}

Response:

{
  "id": 1,
  "username": "test"
}

Foutmeldingen

De API retourneert HTTP-statuscodes om het resultaat van elke aanvraag aan te geven. Hier zijn de mogelijke foutcodes:

Status Code Beschrijving
400 Ongeldige invoer of verzoek
401 Ongeldige gebruikersnaam of wachtwoord
404 Resource niet gevonden
500 Interne serverfout
503 Service niet beschikbaar

Code Voorbeelden

JavaScript (Fetch)

// Inloggen
fetch('http://localhost:8000/api/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'test',
    password: 'password'
  })
})
.then(response => response.json())
.then(data => {
  console.log('Ingelogd als gebruiker:', data);
  
  // Gewicht toevoegen
  return fetch('http://localhost:8000/api/weights', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      user_id: data.id,
      weight: 74.5,
      goal_weight: 70.0
    })
  });
})
.then(response => response.json())
.then(data => console.log('Gewicht toegevoegd:', data))
.catch(error => console.error('Error:', error));

Python (Requests)

import requests

# Inloggen
login_response = requests.post('http://localhost:8000/api/login', json={
    'username': 'test',
    'password': 'password'
})
user = login_response.json()
print(f"Ingelogd als gebruiker: {user}")

# Gewichten ophalen
weights_response = requests.get(f"http://localhost:8000/api/weights/{user['id']}")
weights = weights_response.json()
print(f"Gewichten: {weights}")

# Berekening uitvoeren
calc_response = requests.post('http://localhost:8000/api/calculate', json={
    'gender': 'male',
    'weight': 75.0,
    'height': 180.0,
    'age': 30,
    'activity_level': 'moderate',
    'sport': ['Swimming'],
    'aantal_minuten_sporten': 30,
    'gewenst_gewicht': 70.0,
    'deficit_surplus': 500
})
result = calc_response.json()
print(f"Berekening resultaat: {result}")
